# Khoi's Portfolio Web App

A sleek coffee-themed portfolio built with **Next.js** and **TypeScript**. Managed with **pnpm** and **npm**.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Language:** TypeScript
- **Package Manager:** pnpm / npm
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Planned Features

- Additional framer motion animations
- Discord style window for travels
  - Carousel for each country into 'discord servers' for portolio website
- Animated snoopy hover icon

## Getting Started

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

### 1. Clone the repository

```bash
git clone https://github.com/haidara02/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm i
pnpm i
```

### 3. Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
